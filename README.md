# Everything Store

Page that consumes the <https://fakestoreapi.com/> API and provides sorting, filtering, and search functionality.

## Running Everything Store locally

1. Open the Everything Store project in [VS Code](https://code.visualstudio.com/download)
2. Download the [Live Server](https://marketplace.visualstudio.com/items?itemName=ritwickdey.LiveServer) extension
3. Click to **Go Live** from the status bar
